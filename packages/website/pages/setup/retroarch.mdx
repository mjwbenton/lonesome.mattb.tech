## Basic Configuration Setup

- `Online Updater > Core Downloader` – Download required cores.
- `Input > RetroPad Binds > Port 1 Controls` – Make sure the controls are mapped correctly.
- `Input > Hotkeys` – Assign hotkeys for Menu Toggle, Quit, Fast-Forward, Load State and Save State. Ignore the "(Contoller Combo)" versions of the options.
- `Input` – Ensure "Confirm Quit" is disabled.
- `Saving` – Turn on "Auto Save State", "Auto Load State", "Increment Save State Index Automatically", and ensure "Save Runtime Log (Per Core)" is enabled.
- `Directory` – Override directories for "System/BIOS", "Save Files", "Save States", and "Screenshots" to
- `Configuration` – Turn off "Save Configuration on Quit"

## Per-System Configuration

In general I use core settings rather than content directory settings. This is because I only use one content directory and one core per system,
however some cores cover multiple systems, and in such cases I generally want the same settings for all systems using that core.

- `Video > Scaling` – Enable or disable integer scale, and set to underscale or overscale. Use `Quick Menu > Overrides > Save Core Overrides` to save the setting.
- `Quick Menu > Core Options` – Set relevant core options. These save automatically.
- `Quick Menu > Shaders` – Enable shaders. "Load Preset" selecting the relevant preset from the "shaders_glsl" directory. Then `Save Preset > Save Core Preset`.
