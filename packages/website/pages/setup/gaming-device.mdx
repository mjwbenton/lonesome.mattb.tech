# Gaming Device

## ROM sync script

I have a script to sync ROMs and bios files from my macbook to gaming devices.

[sync_roms.mjs](https://github.com/mjwbenton/mattb-setup.mattbook/blob/master/scripts/sync_roms.mjs)

## Android

### Initial Settings

In Settings App:

- Connect to wifi
- Set timezone
- Turn on Dark theme
- Turn off Floating icon

### Install Obtainium

Download Obtainium from [Obtainium releases](https://github.com/ImranR98/Obtainium/releases)

Download the [Obtainium Emulation Pack](https://github.com/RJNY/Obtainium-Emulation-Pack/releases) and import into Obtainium.
